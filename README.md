## 6 Free AI Developer Tools in One Place

| Tool | What It Does |
|------|-------------|
| **Code Roaster** | Paste code, get roasted at 3 levels: Gentle, Savage, or Absolutely Brutal |
| **Bug Detective** | Paste broken code + error message, get the bug explained AND fixed |
| **Code Explainer** | Explains any code as ELI5, Junior Dev, or Senior Architect |
| **Commit Generator** | Paste your git diff, get 5 perfect commit messages instantly |
| **Interview Prep** | Paste a job description, get 10 technical interview questions + ideal answers |
| **History** | Local history of all your past roasts — your hall of shame |

## Tech Stack

Built with modern, production-grade technologies:

- **Next.js 16** (App Router)
- **Google Gemini AI** (Gemini 1.5 Flash) — The AI that does the roasting
- **Tailwind CSS** + **shadcn/ui** — Beautiful dark UI
- **Vercel** — Lightning fast global deployment

## Who Uses RoastMyCode?

- Developers who want honest code feedback without asking a colleague
- Students who want to understand their code better
- Job seekers preparing for technical interviews
- Teams who need quick commit messages for PRs
